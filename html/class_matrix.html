Detailed Description

A Matrix class using a singleton design pattern.

Parameters
matrixmat
staticMatrix* instance

This class is used to initialize a matrix when opening a file. It uses a singleton pattern becasue after initializing a matrix the program is performing operations only on that particular matrix. That's why we need just one instance of class Matrix.
