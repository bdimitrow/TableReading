Detailed Description

The Formula class is used for editing a cell with a formula.

This class is used to initialize a formula and to read(decipher) it. There are three types of formulas:

  1. number - operator - number ( = 10 ^ 2 );
  2. reference to a cell - operator - number ( = R1C1 / 5 );
    number - operator - reference to a cell ( = 5 * R1C1 );
  3. reference to a cell - operator - reference to a cell ( = R2C2 - R3C4 );
